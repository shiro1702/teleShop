# Dashboard Layout

- Отдельный layout для всех страниц back-office.
- Содержит верхнюю навигацию, контейнер контента, место под сайдбар (когда появится).
- Не использовать для публичной витрины и партнерских страниц.
- В layout допускаются заглушки для будущих разделов с пометкой "в разработке".
