# /:city_slug/:tenant_slug

- Витрина конкретного бренда/ресторана в городе.
- Использовать оба параметра: `city_slug` + `tenant_slug`.
- Проверка, что бренд действительно доступен в выбранном городе.
- Шапка: брендовая и изолированная для ресторана.
- Ссылка «Все рестораны» показывается только здесь (главная ресторана) для планов `basic/pro`.
- На плане `enterprise` в шапке нет ссылок на агрегатор (только юридический футер).
