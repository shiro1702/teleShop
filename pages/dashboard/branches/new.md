# /dashboard/branches/new

- Экран создания нового филиала.

## Что уже можно делать

- Заполнять базовые поля: бренд, город, адрес, часы работы, delivery/pickup.
- Создавать филиал и переходить в его карточку.

## Недочеты / риски

- Не описаны обязательные поля и серверная валидация.
- Нет шаблонов для быстрого создания похожих филиалов.
- Не описаны ограничения по уникальности адреса/названия в рамках города.
