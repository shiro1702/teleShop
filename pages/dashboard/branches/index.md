# /dashboard/branches

- Главный список филиалов организации.

## Что уже можно делать

- Смотреть таблицу филиалов.
- Видеть ключевые поля: бренд, город, адрес, статус, типы получения.
- Выполнять базовые действия: создать, редактировать, архивировать.

## Недочеты / риски

- Не описаны фильтры и поиск по филиалам для больших сетей.
- Не зафиксированы ограничения на архивацию (например, при активных заказах).
- Не описан audit trail по изменениям филиала.
