# /dashboard/branches/:id/zones

- Управление зонами доставки филиала.

## Что уже можно делать

- Настраивать зоны по полигонам/радиусам.
- Указывать цену доставки, бесплатный порог и минимальный заказ.
- Выполнять проверку пересечений зон и смотреть превью расчета.

## Недочеты / риски

- Не описана приоритизация при пересечении нескольких зон.
- Нет проверок на "дыры" покрытия и пограничные точки.
- Не определен UX для сложных геометрий на мобильных экранах.
