# /dashboard/branches/:id

- Карточка конкретного филиала.

## Что уже можно делать

- Смотреть данные филиала.
- Редактировать адрес, расписание, контакты и статус.
- Видеть привязку к бренду и городу.
- Использовать общее расписание ресторана, если у филиала не задан кастомный график.

## Недочеты / риски

- Не зафиксированы ограничения на смену статуса филиала при активных заказах.
- Не описаны поля, редактирование которых доступно только `Owner`.
- Нет явного changelog-а изменений карточки филиала.
- Не описано явно, как переключаться между общим и кастомным расписанием филиала.
