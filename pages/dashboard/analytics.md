# /dashboard/analytics

- Раздел аналитики по продажам и операционным метрикам.

## Что уже можно делать

- Формализовать ключевые метрики и графики для мониторинга.
- Планировать срезы по брендам, городам и филиалам.
- Закладывать экспорт отчета за выбранный период.

## Критичные риски (решаем в MVP)

- [ ] До релиза аналитики зафиксировать единые определения метрик (выручка, заказ, отмена).
- [ ] Проверить доступ к аналитике по ролям (не всем нужен полный срез).
- [ ] Добавить заглушку "в разработке", чтобы не обещать недоступный функционал.

## Некритичные риски (чеклист на потом)

- [ ] Добавить конструктор кастомных отчетов.
- [ ] Добавить расширенные сравнения периодов.
- [ ] Добавить прогнозные метрики и аномалии.
