# /dashboard/menu/modifiers

- Раздел управления модификаторами меню.

## Что должно быть на странице

- Список групп модификаторов:
  - название;
  - тип (`single-choice`, `multi-choice`, `boolean`);
  - область применения (товар/категория);
  - активность.
- Редактор группы:
  - опции внутри группы;
  - доплата на опцию;
  - required/min/max ограничения;
  - сортировка опций.

## User flow (MVP)

1. Пользователь открывает `/dashboard/menu/modifiers`.
2. Создает группу `Размер пиццы`.
3. Добавляет опции: `20см`, `30см`, `40см` с доплатами.
4. Создает группу `Соус` (single-choice).
5. Создает boolean-опции `Добавить сыр`, `Убрать лук`.
6. Привязывает группы к товарам и публикует меню.
