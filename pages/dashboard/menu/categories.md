# /dashboard/menu/categories

- Раздел управления категориями меню бренда.

## Что должно быть на странице

- Таблица/список категорий:
  - название;
  - видимость;
  - порядок отображения;
  - количество товаров внутри категории.
- Действия:
  - создать категорию;
  - переименовать;
  - скрыть/показать;
  - удалить (с проверкой, что нет активных товаров или с переносом).
- Сортировка категорий drag-and-drop (или кнопками вверх/вниз на MVP).

## User flow (MVP)

1. Пользователь открывает список категорий.
2. Нажимает "Создать категорию".
3. Вводит название и сохраняет.
4. Перемещает категорию в нужный порядок.
5. При необходимости скрывает категорию.
6. Сохраняет изменения и публикует меню.
