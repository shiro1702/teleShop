# Карусель: Кальянные / Лаунж-бары (Pocket Menu)

## Аватар сегмента
Темные залы, долгие посадки, динамичный ассортимент.

## Слайды (8 шт.)

1. **Хук:** "В лаунже гость не должен махать руками, чтобы его заметили."
2. Табак и позиции меняются в течение смены, бумажное меню не успевает.
3. Команда тратит время на повторение ассортимента у каждого стола.
4. **Что уже есть в Pocket Menu:** живое QR-меню со стоп-листами в реальном времени.
5. Наличие обновляется сразу для всех столов, без печати и объяснений.
6. Telegram-сценарий вызова персонала: гость дозаказывает без крика через зал.
7. Заказы и обращения приходят структурировано, а не "кто-то звал".
8. **CTA:** "Напишите `КАЛЬЯН` — настроим рабочий сценарий для вашего зала."

## Подпись к посту
В лаунж-формате решает скорость реакции и точность коммуникации.  
Pocket Menu дает гостю тихий и понятный канал заказа, а команде — порядок вместо хаоса.  
Меньше потерянных столов и больше повторных заказов за вечер.

## Тезисы для комментариев/Директа

- "Подходит для темных и шумных залов."
- "Гостю удобно, персоналу прозрачно."
