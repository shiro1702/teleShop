# Instagram-карусели для первых B2B клиентов (Pocket Menu)

Этот пакет собран на основе диалога в `docs/chat/питч для b2b.md`, тарифных наработок из `docs/chat/тариф.md` и модульной модели из `docs/FEATURE_TOGGLES_PRICING_RU.md`.

## Цель

- Привлекать первых клиентов через конкретные боли сегментов, а не через абстрактные "цифровые решения".
- Продавать уже реализованный функционал Pocket Menu: QR, Telegram-заказы, режимы заказа, delivery-зоны, CRM-модули.
- Снимать страх внедрения через консьерж-онбординг: команда Pocket Menu запускает все под ключ.

## Структура пакета

- `segment-1-streetfood.md` — киоски, шаурма, кофе to-go.
- `segment-2-coffee-bakery-cafe.md` — кофейни, пекарни, кафе.
- `segment-3-restaurants-gastro.md` — рестораны, гастробары, лаунж.
- `segment-4-hookah-lounge.md` — кальянные и лаунж-бары.
- `segment-5-bars-clubs.md` — шумные бары, пабы, клубы.
- `segment-6-dark-kitchens-delivery.md` — dark kitchen и домашние кондитеры.
- `segment-7-hotels-saunas-recreation.md` — базы отдыха, гостиницы, сауны, бильярдные.
- `public-roadmap.md` — общий roadmap для демонстрации амбиций продукта.

## Единые правила подачи каруселей

1. Слайд 1: одна острая боль сегмента.
2. Слайды 2-3: что бизнес теряет в деньгах/скорости/качестве.
3. Слайды 4-6: каким конкретно функционалом Pocket Menu это закрывается.
4. Слайд 8: CTA с простым первым шагом (демо/пилот/подключение).

## CTA-модель (для всех постов)

- "Напишите в Директ: `ДЕМО`"
- "Запустим за 24 часа без вашей рутины"
- "Вы даете меню, мы делаем остальное"

## Контент-план публикаций (быстрый старт)

- Неделя 1: сегменты 1, 2, 3 (массовый рынок + быстрые сделки).
- Неделя 2: сегменты 4, 5 (ночные форматы).
- Неделя 3: сегменты 6, 7 (доставка и hospitality).
- Неделя 4: `public-roadmap.md` + пост "что уже работает в Pocket Menu сегодня".
