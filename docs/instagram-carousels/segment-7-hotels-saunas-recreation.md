# Карусель: Базы отдыха / Гостиницы / Сауны / Бильярд (Pocket Menu)

## Аватар сегмента
Еда — доп. сервис, который часто недопродается из-за неудобного заказа.

## Слайды (8 шт.)

1. **Хук:** "Если заказ еды требует звонка — часть гостей просто не закажет."
2. Администратор занят и не всегда быстро принимает заявки.
3. В итоге проседает допродажа еды и напитков в номерах/домиках.
4. **Что уже есть в Pocket Menu:** QR-коды под каждую локацию.
5. Гость сканирует код и заказывает в Telegram без звонка.
6. Кухня сразу получает, куда нести: номер, домик, зал.
7. При необходимости включаете самовывоз/доставку в рамках объекта.
8. **CTA:** "Напишите `БАЗА` — запустим удобный room-service без колл-центра."

## Подпись к посту
В объектах отдыха выигрывает сервис, который работает "в один скан".  
Pocket Menu превращает хаотичный room-service в понятный канал дополнительной выручки.  
Гостям удобнее, команде проще, заказов больше.

## Тезисы для комментариев/Директа

- "Запуск поэтапный: сначала QR-меню, затем полный заказ."
- "Подходит для номеров, домиков, саун и приватных зон."
- "Внедрение без перегруза администраторов."
