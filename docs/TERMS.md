# Словарь терминов TeleShop

Этот словарь фиксирует единообразие терминов в продукте, документации и интерфейсах.

## Базовое правило

- Во внутренних разделах дашборда писать: **«Филиал»**.
- В клиентской витрине писать: **«Ресторан»**.

## Применение по зонам

- `dashboard/*`, onboarding партнера, внутренние CRUD-формы:
  - «Филиал», «Список филиалов», «Создать филиал», «Настройки филиала».
- Публичные страницы витрины (`/:city_slug/...`, `/:city_slug/:tenant_slug/...`), пользовательские тексты:
  - «Ресторан», «Выберите ресторан», «Меню ресторана», «Доставка ресторана».

## Техническая ремарка

- В БД и API используется таблица `restaurants`.
- В интерфейсе дашборда это отображается как «Филиалы».
- В интерфейсе витрины это отображается как «Рестораны».
